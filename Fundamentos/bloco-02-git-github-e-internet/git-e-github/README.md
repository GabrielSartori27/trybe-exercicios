# DIA 1 E 2

Os exercícios desse dia foram sobre a utilização dos principais comandos relacionados ao git e github. Aprendemos a utilizar o git init, git branch, git checkout/git checkout -b, git merge, git clone, git pull, git add ., git commit -m, git push entre outros. Em suma, aprendemos todos os comandos necessários para criar um repositótio git remoto e local e sincronizar os dois.

As pastas referentes aos exercícios desses dois dias se encontram vazias pois eles consistiam na criação de um repositório no github e sincronização dos dados com o repositório local. Sendo assim, os resultados deles podem ser observados ao conseguir acessar meu github e viusalizar meus repositórios. Todas essas etapas foram parte dos exercícios desses dias.
